# CF & Physio Research Hub

## DEPLOY IN 30 SECONDS (after downloading files):

Open Terminal (Mac) or Command Prompt (Windows) and run:

```bash
cd ~/Downloads/physioandnews
git init
git remote add origin https://YOUR_TOKEN@github.com/kmwood45-ctrl/physioandnews.git
git checkout -b main
git add .
git commit -m "update"
git push -f origin main
```

Replace YOUR_TOKEN with a fresh token from https://github.com/settings/tokens
(check the "repo" box when creating it)

## GitHub Pages (first time only):
